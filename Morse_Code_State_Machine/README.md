CS 350 Milestone Two: GPIO UART Lab

This project was completed during Module Two of CS 350 and focuses on using
UART communication to control hardware on a Raspberry Pi. The project uses
a simple client–server setup where a Python client sends ON, OFF, and EXIT
commands to a server program, which then turns an LED on or off using GPIO.

A state machine is used to manage how the system responds to user input and
controls the flow of the application. Creating the state machine diagram
helped me better understand how the system transitions between states and
kept the implementation organized.

This project helped strengthen my understanding of UART communication,
GPIO control, event-driven programming, and how software can interact with
physical hardware in an embedded system.
