Smart Thermostat Final Project – CS 350

This project was completed as the final project for CS 350 and builds on the
embedded systems concepts introduced throughout the course. The goal of the
project was to design and implement a smart thermostat using a Raspberry Pi
that could respond to temperature readings and user input.

The thermostat uses a temperature sensor to read the current temperature and
allows the user to cycle between off, heat, and cool modes using buttons.
LEDs provide visual feedback for each mode, and a 16x2 LCD displays the
current time, temperature, and thermostat state. A state machine is used to
manage the system’s behavior and ensure each mode responds correctly based
on temperature and user actions.

Working on this project helped me better understand how software and hardware
work together in an embedded system. It reinforced the importance of planning
state transitions ahead of time, debugging incrementally, and writing
organized code that can be maintained and expanded in the future.
